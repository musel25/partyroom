# partyroom

Self-hosted YouTube watch-party — synced playback, chat, queue, and reactions, with a Duolingo-inspired UI.

🌐 **Live:** [partyroom.musel.dev](https://partyroom.musel.dev) *(coming soon)*

## What it does

- Paste any YouTube link, share a room link with friends.
- Everyone watches in perfect sync (play, pause, seek, skip — democratic control).
- Chat in real time. Add videos to a shared queue. Send floating-emoji reactions.
- Sign in with Google for friends + room history, or join as a guest with just a nickname.

## Stack

Next.js 15 · TypeScript · Tailwind · shadcn/ui · Socket.IO · PostgreSQL · Prisma · Auth.js v5 (Google OAuth) · Docker · Caddy
